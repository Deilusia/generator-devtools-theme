## Getting started

1. Install using NPM:

    ```sh
    $ [sudo] npm install -g generator-devtools-theme
    ```

2. Scaffold your new project:

    ```sh
    $ yo generator-devtools-theme
    ```

3. Start writing your Chrome DevTools Theme!
